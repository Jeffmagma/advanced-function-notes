	\subsubsection{Relations}
	A (binary) relation is defined as a set of ordered pairs $(x, y)$.
	A relation can be described using:
	\begin{itemize}
		\item words
		\item graphs
		\item equations
		\item inequalities
		\item lists of ordered pairs
		\item mapping diagrams 
	\end{itemize}
